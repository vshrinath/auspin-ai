# Salient Platform Template

A production-ready monorepo template with:

- 200+ Tailwind UI components
- Next.js + Astro + FastAPI architecture
- AI agent guidelines for rapid development
- Automated component import system

## Quick Start

1. Use this template to create a new repository
2. Clone and install dependencies: `npm install`
3. Start development: `npm run dev --workspace web`
4. Browse components: Visit `/test-components`

## What's Included

- **Complete Tailwind UI Library**: 45+ sections, 200+ components
- **Multi-App Architecture**: Web (Next.js), Docs (Astro), API (FastAPI)
- **AI Development Guidelines**: Page composition and API design
- **Pattern Import System**: Automated component generation
- **Production Ready**: TypeScript, testing, proper tooling
